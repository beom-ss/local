HTML

자주 사용하는 tag에 대한 설명

p는 문단을 나누는 tag입니다.
css를 사용해서 문단 사이의 간격도 조정할 수 있습니다.

strong은 굵게 표시하는 tag입니다.
u는 밑줄을 표시하는 tag입니다.
h는 제목을 표시하는 tag입니다.
ol은 순서가 있는 목록을 표시하는 tag입니다. ordered list
ul은 순서가 없는 목록을 표시하는 tag입니다. unordered list
li는 목록의 항목을 표시하는 tag입니다. list item
table은 표를 표시하는 tag입니다.
tr은 표의 행을 표시하는 tag입니다. table row
td는 표의 열을 표시하는 tag입니다. table data

진짜 중요한 tag들!!

html은 html 문서의 시작과 끝을 표시하는 tag입니다.
head는 html 문서의 정보를 표시하는 tag입니다.
body는 html 문서의 내용을 표시하는 tag입니다.
a는 링크를 표시하는 tag입니다. anchor
